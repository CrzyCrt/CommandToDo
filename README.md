# CommandToDo
Description:
A simple command-line to-do list application that allows users to manage a to-do list file.

Features:
> Command-Line UI
> Read from and save to file
> Immediate feedback at each step

How to run:
python main.py

What I Learned:
> File Handling
> Conditional Statements
> User Input

Future Improvements:
> Convert to GUI with more advanced selection/display options
  > Add checkboxes to displayed list to handle completed tasks
> Add motivaional messages when user progresses through completed_task milestones
